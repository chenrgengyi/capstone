# EDS Captone

This project analyzes a primary spatial dataset consisting of camera trap station locations and species encounter records from the North China Leopard "Leopard Country Field" project in Shanxi, China.

## Project Structure
- data/raw: Original spatial and species encounter data
- data/processed: Cleaned and processed joint dataset
- scripts: Analysis scripts (in R)
- docs: Documentation and notes
- figures: Generated plots and visualizations

Datasets used:
Dataset 1: Leopardfield Camera Station Shapefile (Spatial coordinates for monitoring points).
Dataset 2: Species Encounter Ledger (Long-form observation data for fox and other wildlife species).
